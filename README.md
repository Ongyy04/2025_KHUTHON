## 🌱 도시 농사를 손안에서 즐기는 가상 농사 플랫폼 SEEDO

#### 2025 KHUTHON 최우수상, 인기
#### 개발기간 : 2025.05

## 👋🏻 프로젝트 소개

도시에서는 농작물을 키우기 어렵다는 현실 속에서도,
누구나 쉽게 씨앗을 심고, 작물을 키우고, 수확의 성취감을 경험할 수 있다면 어떨까요?

SEEDO는 가상 농사와 게이미피케이션을 결합한 웹 서비스로,
사용자가 씨앗을 심고 작물을 키우며
수확한 농작물을 보관·판매하거나 실제 배송까지 이어지는 경험을 제공합니다.

농사의 진입 장벽을 낮추고,
편리함 속에서 성장과 보상의 즐거움을 느낄 수 있도록 설계된
도시형 가상 농사 플랫폼입니다.
